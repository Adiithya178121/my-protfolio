Task Manager Web App

A secure full-stack task manager with Firebase Authentication and REST API operations.

  • User Login/Signup
  • CRUD for tasks (Add, Edit, Delete, View)
  • Filters & Sorting features
  • Protected Dashboard using Firebase UID

Deep Explanation

Problem: Users need a secure place to store and manage tasks online.

Tech Stack: HTML, CSS, JS, Firebase Auth, Firebase DB, Fetch API, Vercel.

Architecture: User logs in → Firebase returns UID → tasks stored under UID → CRUD operations via REST API.

Key Logic: Tasks saved as JSON under tasks/UID/taskID. Each user can only access their data.

Features: Auth, Task creation, Editing, Deleting, Sorting, Filtering.

Challenges solved: User session handling, data isolation, async fetch management.

Learnings: Authentication, REST APIs, state management, cloud deployment.

Product Explorer Web App

An API-based product browsing application with pagination, sorting, and filtering.

  • Pagination (4×2 Grid)
  • Category filters
  • Sorting by price & title
  • Debounced search bar

Deep Explanation

Problem: E-commerce UI requires sorting, searching, filtering, and page navigation.

Tech Stack: HTML, CSS, JS, FakeStore API, LocalStorage, URL Params, Vercel.

Architecture: Fetch products → render 8 items/page → store page state in URL/localStorage.

Pagination: Uses slice() to show items for each page.

Sorting: Sort data by price and title before rendering.

Filtering: Filters by categories such as electronics, fashion, etc.

Debouncing: Reduces API calls by delaying search.

Challenges solved: Combining sort+filter+pagination and optimizing search.

Learnings: API handling, performance optimization, UI state control.
